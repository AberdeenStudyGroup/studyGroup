#### Use R’s merge function to join forms

```
parent <- read.csv(‘file_path’, check.names=F)
child <- read.csv(‘file_path’, check.names=F)

tutorial <- merge(parent, child,
by.x=‘ec5_uuid’, by.y= ‘ec5_parent_uuid’)
```
